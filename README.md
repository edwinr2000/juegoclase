
# Tirador Espacial con PyGame

## Descripción
Este es un juego simple de disparos espaciales creado con la librería PyGame. Controla una nave, dispara a los enemigos y acumula puntos. Evita chocar con los enemigos y asegúrate de que no lleguen al final de la pantalla.

Desarrollado por Edwin Oswaldo Torres Rincón

## Instrucciones

### 1. **Instalar las dependencias**
   Antes de comenzar a jugar, asegúrate de tener la librería PyGame instalada. Puedes hacerlo ejecutando el siguiente comando en tu terminal o consola de comandos:
   
   ```bash
   pip install pygame
   ```

### 2. **Iniciar el juego**
   - **Seleccionar el nivel de dificultad**: Al inicio del juego, se te pedirá que elijas un nivel de dificultad entre **Fácil**, **Medio** o **Difícil**. Haz clic en el nivel que prefieras.
   - **Escribir tu nombre**: También se te pedirá que ingreses tu nombre en el campo correspondiente. Asegúrate de escribirlo correctamente.
   - **Presionar Enter**: Cuando hayas seleccionado el nivel de dificultad y escrito tu nombre, presiona la tecla **Enter** para comenzar el juego.
   - **Reiniciar el juego**: Cada vez que reinicies el juego, asegúrate de que tu nombre esté escrito y el nivel de dificultad seleccionado antes de presionar Enter.

### 3. **Cómo jugar**
   - **Movimiento**: Usa las teclas **izquierda** y **derecha** para mover tu nave de un lado a otro.
   - **Disparo**: Pulsa la tecla **espacio** para disparar balas a los enemigos.
   - **Evitar colisiones**: Asegúrate de que los enemigos no toquen tu nave, ya que perderás una vida si esto sucede.
   - **Puntaje**: Cada vez que destruyas un enemigo, ganarás puntos. ¡Intenta lograr la mayor cantidad de puntos posible!

### 4. **Controles adicionales**
   - **Play/Pause Música**: En el menú de inicio, puedes hacer clic en el botón **Música SI/NO** para alternar el sonido de fondo del juego.

### 5. **Objetivo del juego**
   - El objetivo es destruir a los enemigos que caen del espacio y evitar que lleguen al fondo de la pantalla.
   - Si tus vidas se acaban, el juego terminará y podrás ver tu puntaje final.

### 6. **Reiniciar el juego**
   - Después de perder todas tus vidas, haz clic en cualquier parte de la pantalla para reiniciar el juego.
