# Shooter Espacial

## Descripción
Este es un juego simple de disparos espaciales creado con la librería PyGame. Controla una nave, dispara a los enemigos y acumula puntos.

Vete moviendo y disparando para evitar chocar con los enemigos, y evitar que los enemigos lleguen al final.
